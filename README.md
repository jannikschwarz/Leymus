# Leymus
AND application 
This application is made for the Leymus art studio near Nørre nebel. The requirements are as following
1. As a user i want to be able to view the newest update at Leymus Art studio
2. As a user i want to be able to view the gallery 
3. As a user i want to be able to create a Lesson for the art Studio
4. As a user i want to be able to view my own Lessons that i have ordered
