# Leymus
AND application This application is made for the Leymus art studio near Nørre nebel. The requirements are as following

As a user i want to be able to view the newest update at Leymus Art studio
As a user i want to be able to view the gallery
As a user i want to be able to create a Lesson for the art Studio
As a user i want to be able to view my own Lessons that i have ordered
The application purpose is to enable anyone who is interested in the art gallery to download the app and overview the current situation of the gallery. This can be done by looking at the news panel to see any update of the gallery, the main page to see offers and ofc join the newsletter by following the link provided. On top of that they can also order a lesson that they want to peform in the gallery's many workshops.
